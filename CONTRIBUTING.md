# Contributing to spawn

## Setup

```bash
git clone https://github.com/konard/spawn
cd spawn
pnpm install
pnpm run spawn --help
```

## Development

```bash
pnpm run dev        # watch mode
pnpm run build      # build all packages
pnpm run test       # run tests
pnpm run lint       # lint
```

## Commit convention

```
feat(core): add new feature
fix(shell): fix bug
chore(rebrand): maintenance task
```
